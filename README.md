# Crypto Trading Agent

## Elevator Pitch

Self-improving AI trading agent. Learns from each trade through Think-Act-Eval-Feedback to get smarter over time.

## Self-Improving Loop

The agent operates on a simple feedback loop:

```
Think → Act → Eval → Feedback → Update → Repeat
```

### Think
- Research and gather all relevant information
- Analyze market conditions and price data
- Consider historical performance

### Act
- Make a decision: **Buy** or **Sell**
- Execute the trading action

### Eval
- Evaluate the outcome: Did the decision result in **profit** or **loss**?
- Measure performance against the goal

### Feedback
- Root Cause Analysis (RCA) of failures
- Identify what went wrong and why
- Generate insights for improvement

### Update
- Refine the decision-making process
- Update prompts and strategies
- Learn from mistakes

### Repeat
- Continue the cycle with improved knowledge
- Iteratively get better over time

## Quick Start

1. Install dependencies
2. Set up environment variables
3. Run the agent
4. Watch it learn and improve
